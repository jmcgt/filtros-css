• Filtros CSS

    Alguns filtros para estilizar suas fotos sem deixá-las mais pesadas e sem Photoshop:

  • Preto e Branco

                           
                       .grayscale
                       {
                          filter: grayscale(100%);
                       }

  • Saturação

                           
                       .saturate
                       {
                            filter: saturate(150%);
                       }

  • Sépia

                           
                       .sepia
                       {
                            filter: sepia(100%);
                       }

  • Desfoque

                           
                       .blur
                       {
                            filter: blur(2px);
                       }

  • Brilho e Contraste

                           
                       .brightness-contrast
                       {
                            filter:brightness(150%) 
                                   contrast(150%);
                       }

  • Negativo

                           
                       .invert
                       {
                            filter: invert(100%);
                       }
